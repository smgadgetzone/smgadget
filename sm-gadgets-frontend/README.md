# SM Gadgets - Premium Mobile Accessories Store

Welcome to the official SM Gadgets repository. This platform is a full-featured e-commerce solution for premium mobile accessories.

## Features
- Premium UI/UX with React & Tailwind CSS
- Secure Razorpay Payment Integration
- Real-time Inventory Management
- Personalized Coupon System
- Order Tracking & Professional Email Updates

## Technology Stack
- **Frontend**: Vite, React, TypeScript, shadcn-ui, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB Atlas
- **Payments**: Razorpay
- **Emails**: Nodemailer

## Development

```sh
# Install dependencies
npm i

# Start development server
npm run dev
```

## Deployment
The project is configured for seamless deployment on **Vercel** (Frontend) and **Render** (Backend).
